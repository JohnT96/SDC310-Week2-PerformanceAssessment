# SDC310 Week 2 Performance Assessment

This project is a PHP form that collects user input and displays
submitted values or error messages if fields are left blank.

## Technologies Used
- PHP
- HTML
- XAMPP (Apache)

## How to Run
1. Place file inside xampp/htdocs directory
2. Start Apache in XAMPP
3. Navigate to:
   http://localhost/foldername/Tenney_Wk2_Performance_Assessment.php
